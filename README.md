# 🛒 Aling Nena Grocery Store
**Course Project: Website & Technologies-1 | Group 6 | IT-1P | Philippines**

# Description

Aling Nena’s Online Grocery Store is a specialized e-commerce platform designed to bring the traditional Filipino "Sari-Sari Store" 
experience into the digital age. Specifically serving the community of Muntinlupa, this store allows local residents to browse daily essentials—from delata (canned goods) to household appliances—through a user-friendly online interface.

---

## 🎯 Project Goals
* **User Experience:** Provide a clean, modern website interface for all users.
* **Functionality:** Implement a working navigation system, product categories, and a user feedback/Shopping/login portal.
* **Compatibility:** Ensure full responsiveness across mobile devices, specifically fixing layout issues in **landscape orientation**.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for the store and forms.
* **CSS3:** Custom styling featuring:
  * Flexbox and CSS Grid for layout.
  * Media Queries for mobile and landscape optimization and compatibility.
* **Font Awesome:** For interactive icons (Search, Cart, User).
* **JavaScript:** To handle mobile & desktop menu toggles swiping function, and form interactions.
